O programa deve ser executado da seguinte forma:

1 - Download da ferramenta XAMPP para o sistema operacional (Foi utilizado o Windows neste exemplo)
2 - No seu local de Instalação, haverá uma pasta XAMPP e dentro dela uma subpasta chamada htdocs. Em htdocs, deve-se colocar uma cópia deste repositório.
3 - Execute o aplicativo XAMPP e inicialize o Apache e o MySQL
4 - Para importar a base de dados, em um navegador web ou utilizando um sgbd de qualquer preferência, restaure a base de dados MySQL bd2.sql que se encontra dentro da pasta Documentos deste repositório.
5 - Para acessar a aplicação, você deve acessar em seu navegador: localhost/camille_consulta. (Este camille_consulta é referente ao nome da pasta em que você colocou em htdocs, caso você tenha renomeado o arquivo, deve mudar a URL de acesso).
